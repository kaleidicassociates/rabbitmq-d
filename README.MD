# rabbitmq-d - Beta binding and high-level D wrapper for librabbitmq-c.

[![Build Status](https://travis-ci.org/symmetryinvestments/rabbitmq-d.png?branch=master)](https://travis-ci.org/symmetryinvestments/rabbitmq-d)
